
# Face-Mask-Detector

This project aims to provide an understanding of how Computer Vision works. This model is trained to focus on a plethora of images and detect if people are wearing agreed upon COVID-19 masks properly, improperly or not at all.



## Deployment
To run this project, simply clone this repo and run on Python. Make sure the latest versions of OpenCV is installed.


## Tech Stack
1. Python
2. Tensorflow
3. OpenCV
4. XML
